2. forEach()
forEach 메서드는 다음 매개변수(parameter)와 함께 배열의 각 요소에 적용하게 될 콜백 함수(callback function)를 전달한다.

Current Value (명명된 매개변수) - 처리할 현재 요소
Index (선택적 매개변수) - 처리할 현재 요소의 인덱스
Array (선택적 매개변수) - forEach 메서드를 호출한 배열

=> 매개변수로 함수를 받는다. (콜백함수)
=> 자체 반환값은 없다.
=> 복사된 값은 새로운 배열이다.
