순수 함수 => 인자에 같은 값을 넣으면 항상 같은 값을 반환하는 함수
	=>  함수 외부의 조작으로 아무것도 바뀌지 않는다
	=> 아무런 부수 효과도 트리거하지 않는다.

부수 효과 => 함수의 외부를 바꿔놓는 것을 의미한다.
	=> 함수의 밖에서 정의된 변수를 바꾸는 행위나 함수에 들어가는 객체나 배열을 바꾸는 행위 등...
	=> http 요청으로 값을 가져오거나 바꾸거나, db의 데이터를 사용하거나 


팩토리 함수 => 내부 함수는 외부 함수의 매개변수나 변수에 접근할 수 있고 그 받은 값을 닫힌 후에도 기억할 수 있다. 이를 이용한 함수이다.


closure 
중첩 함수 => 함수 안에 함수가 존재하는 함수를 의미한다. 중첩함수이면서 함수 반환일 수 있고 중첩함수만일 수 있다.
함수 반환 => 함수 안에 함수가 존재하고 이를 return 하는 것이다
중첩 함수만일 때는 호출을 하는 것이고 중첩함수이면서 함수 반환이면 return을 하는 것이다. 함수 호출이나 return은 내부 함수 scope에 해야 한다.

클로저라는 의미는 외부 함수가 닫힌 이후에도 내부 함수는 이를 기억하고 사용할 수 있다는 것이다.


재귀 함수 
함수내에서 자신의 함수를 호출하는 것을 의미한다. => 스탑로직이 없다면 무한 로프를 돈다.
수많은 객체나 배열로 덮힌 것을 가져올 떄 유용하게 사용된다.
