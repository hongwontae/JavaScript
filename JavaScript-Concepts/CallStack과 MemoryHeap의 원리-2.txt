1. 변수
=> 하나의 값을 저장하기 위해 확보한 메모리 공간 자체, 메모리 공간 식별하기 위해 붙인 이름

2. 할당
=> 변수에 값을 저장하는 것을 의미한다.
=> 할당 연산자를 통해 할당한다. (=)

3. 참조
=> 변수에 저장된 값을 읽어 들이는 것을 의미한다.

4. 선언
=> 변수명을 JS 엔진에게 알리는 것을 선언이라고 한다.
=> var, let, const 키워드를 통해 가능하다.

=> JS에서 선언할 떄 크게 두 가지 과정을 거친다.
1. 선언 단계 : 변수명을 JS 엔진에게 알린다.
2. 초기화 단계 : 값을 저장하기 위한 메모리 공간 확보, undefined 할당


5. 선언과 할당
=> 선언과 할당은 하나의 문으로 단축할 수 있지만 실행 시점이 다르다.
=> 변수 선언은 런타임 전에 호이스팅되어 실행, 할당은 런타임에 실행된다.
