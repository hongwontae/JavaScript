1. Hosting
1-1. 자바스크립트가 실행 전에 변수/함수선언을 위로 끌어올리는 동작입니다.
1-2. JS 엔진이 내부적으로 먼저 선언을 인식해서 메모리에 올려둡니다.
1-3. ** 정확히 그 대상은 함수 선언문과 var 변수 선언이 대상입니다. **

2. 예시-1
 var a;
 console.log(a); Reference Error가 아닌 undefined
 a=10;
=> 즉, var로 선언한 변수는 호이스팅되어 undefined가 할당됩니다.

3. 예시-2
 say() -> HI가 출력
 function say(){
    console.log('HI');
 }
=> 함수 선언문은 전체가 위로 끌어올려집니다.


4. TDZ(Temporal Dead Zone, 일시적 사각 지대)
4-1. let과 const는 선언 전에 접근하려고 하면 ReferenceError가 발생하는 구간이 있습니다.
=> 이 구간은 TDZ라고 합니다.
4-2. 즉, let과 const로 선언한 변수는 호이스팅되지만 초기화되지 않고 선언 ~ 초기화 이 중간 구간을 TDZ라고 합니다.
