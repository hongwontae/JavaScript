1. var, let, const는 변수 선언 키워드라고 불립니다.

2. var
2-1. 함수 스코프입니다.
=> 함수 내부에서 var로 선언한 변수는 오작 그 함수 내부에서만 유효합니다.
2-2. 재할당이 가능합니다.
2-3. 재선언이 가능합니다.
2-4. 초기화 없이 선언이 가능합니다.


3. let
3-1. 블록 스코프입니다.
=> let 키워드로 선언한 변수는 오직 그 블록 내에서만 유효합니다.
3-2. 재할당이 가능합니다.
3-3. 재선언은 불가능합니다.
3-4. 초기화 없이 선언이 가능합니다.


4. const
4-1. 블록 스코프입니다.
4-2. 재할당과 재선언이 불가능합니다.
4-3. 초기화 없이 선언도 불가능합니다.
