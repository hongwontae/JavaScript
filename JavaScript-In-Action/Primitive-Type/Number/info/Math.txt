1. Math.round()
=> 반올림 함수

2. Math.floor()
=> 내림 함수

3. Math.ceil()
=> 올림 함수

4. Math.trunc()
=> 소수점을 버리고 정수만 반환하는 함수

5. Math.abs()
=> 절대값 반환 함수

6. Math.sign()
=> 양수면 1, 음수면 -1, 0이면 0을 반환하는 함수

7. Math.max(...args)
=> 최대값을 반환하는 함수

8. Math.min(...args)
=> 최소값을 반환하는 함수

9. Math.pow(x,y)
=> x의 y 제곱
=> x가 2이고 y가 3일 떄 2의 3승을 반환합니다. => 8

10. Math.sqrt(x)
=> 루트

11. Math.random()
=> 0이상 1미만의 난수를 반환합니다.
=> Math.floor(Math.random() * 10) + 1; -> 1이상 10이하의 정수를 만드는 방법입니다.
