1. Set
=> 중복되지 않는 값들의 집합입니다.
=> 요소의 순서가 삽입 순서입니다.
=> 배열처럼 반복이 가능합니다.
=> set 객체를 하나씩 거내면 그냥 일반 요소입니다.


2. set 생성방법
2-1. new Set([...n])
=> 이터러블한 객체를 모두 인자로 받을 수 있습니다.

2-2. .add(value)
=> 이 떄 중복된 값은 무시합니다.


3. [...new Set([1,1,1,2,2,2])]
=> 중복된 값을 제거하고 일반 배열이 만들어집니다.

4. set.has(value)
=> 해당 값이 존재하는지 확인, true or false

5. set.delete(value)
=> 특정 값을 삭제합니다.

6. set.clear()
=> 모든 값을 삭제합니다.

7. set.size()
=> 요소 개수를 반환합니다.

8. for-of, forEach, Spread 연산자를 사용할 수 있습니다.
