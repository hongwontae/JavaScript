자바스크립트는 싱글 쓰레드 언어이다.(한 번에 한가지 일만 처리한다. alert()을 생각하면 된다.)

JS엔진은 코드를 이해하고 실행을 도와주는 역할을 한다.
JS엔진은 emory Heap과 Call Stack으로 이루어져 있다.

Memory Heap
데이터를 임시 저장하는 곳으로, 함수나 변수, 함수를 실행할 때 사용하는 값들을 저장합니다.

Call Stack
코드가 실행되면 코드의 내부의 실행 순서를 기록해 놓고, 하나씩 순차적으로 진행할 수 있도록 도와주는 곳입니다.

call stack에 쌓인 코드 때문에 특정 이벤트를 실행하지 못하는 경우가 발생한다. => 이벤트 루프를 사용한다.
