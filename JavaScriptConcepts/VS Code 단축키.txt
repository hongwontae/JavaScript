1. ctrl+tab => 창 이동

2. ctrl+w => 창 닫기

3. ctrl+p => 파일 열기

4. ctrl+q => 메뉴 열기

5. ctrl+->/<- 단어 단위로 커서 이동: Ctrl + Left Arrow / Ctrl + Right Arrow

6. ctrl+l 라인 전체 선택: Ctrl + L

7. ctrl+shift+k => 라인 전체 삭제

8. 명령 팔레트 => ctrl+shift+p

9. ctrl+p => 전체 디렉토리에서 파일명 검색

10. ctrl+shift+tab => 열려있는 파일 이동하기
