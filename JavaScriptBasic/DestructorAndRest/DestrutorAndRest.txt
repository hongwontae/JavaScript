...은 전개연산자나 나머지 매개변수로 사용된다.
함수 호출 시 인자에 사용되는 ...은 배열을 목록으로 확장해주는 전개 연산자이다.
함수 매개변수에 사용되는 ...은 인수 목록의 나머지를 배열로 모아주는 나머지 매개변수이다.
일반적인 let [...b1] = b; => 나머지 연산자를 사용한 것이다.

객체에 ...은 전개 연산자로 사용된다. 객체를 목록을 확장해주는 것이다.


구조분해할당
배열이나 객체에서 사용될 수 있다.

let a = [1,2,3];
let [a1,a2,a3] = a;
a1 =1, a2 =2, a3=3 => 이렇게 값을 분해하는 형식을 말한다.

let person {
    name : 'hwt',
    age : 30,
}
let {name,age} = person;
name = 'hwt', age = '30'; => 이렇게 값을 분해하는 형식을 말한다.
객체를 구조분해할떄는 키값이 변수명이 되도록 설정해야 한다.

구조분해할당은 기본값을 줄 수 있다.
객체 구조분해할당은 기본값과 변수명을 줄 수 있다. => 객체의 변수명은 기본적으로 키 값이기 떄문이다.
