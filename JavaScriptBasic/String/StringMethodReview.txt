1. charAt(); => 인덱스에 해당하는 문자열을 반환한다.

2. indexOf(); => 문자열에 해당하는 인덱스를 반환한다. => 중복된 문자열일 경우 두 번쨰 매개변수를 활용한다.
    (두 번쨰 인자는 해당 인덱스 이후부터 indexOf()로직을 실행하라라는 뜻이다.)
    
3. lastIndexOf(); => indexOf()와 같은 로직이지만 첫 번째 인자로 끝날 떄는 음수를 사용할 수 있다.
    (음수는 뒤에서부터 시작한다라는 의미이다.)

4. replace(); => 첫 번쨰 인자에 해당하는 문자열을 두 번쨰 인자에 해당하는 문자열로 바꾸라는 메서드이다.

5. repeat(); => 인자에 숫자를 주면 그 숫자만큼 반복하라는 메서드이다.

6. substring(); => 원하는 문자열만 가져오고 싶을 떄 사용하는 메서드이다.
    (첫 번쨰 인자는 시작점 인덱스, 두 번쨰 인자는 종점 인덱스/ 인자가 하나라면 시작점 인덱스에서부터 모조리 가져온다.)

7. slice(); => substring과 같지만 음수로직을 사용할 수 있다.

8. startsWith(); => 해당 인자로 문자열이 시작하는지 Boolean 타입으로 반환한다.

9. endsWith(); => 해당 인자로 문자열이 끝나는지 Boolean 타입으로 반환한다.

10. valueOf(); => 객체값을 원시값으로 바꿔주는 메서드이다.

11. includes() => 해당 문자열이 있는지 Boolean 타입으로 반환한다.

12. split() => 문자열을 배열로 바꾸는 메서드이다.
    (첫 번쨰 인자는 구분자이다. 구분자는 배열의 값에서 제거된다. 두 번쨰 인자는 총 개수를 제외하는 로직이다.)
