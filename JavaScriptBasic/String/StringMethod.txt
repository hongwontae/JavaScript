String은 원시값이다. => 스트링은 불변이다. => 스트링 메서드는 항상 새로운 값을 반환한다.
String은 기본적으로 원시값이지만 참조값으로 설정할 수 있다. => let a = new String('Salah');

String Method

1. charAt();
=> 인자의 인덱스에 대한 문자열을 반환한다.


2. indexOf();
=> 인자의 문자열에 대한 인덱스를 반환한다.


3. lastIndexOf();
=> IndexOf와 같은 논리지만 뒤에서부터 시작한다.


4. replace();
=> 첫 번쨰 인자는 해당할 문자열, 두 번쨰 인자는 바꿀 문자열


5.split();
=> 문자열을 배열로 만드는 메서드
=> 인자에 구분자를 주어서 쪼꺠서 넣을 수 있다. 구분자는 추가되지 않는다.


6. subStirng();
=> 인자의 인덱스를 분석해서 문자열을 가져오는 메서드이다.
=> 인자를 넣지 않으면 전체를 복사하여 반환한다. => 깊은 복사이다.
=> 인자가 하나라면 해당 인덱스부터 모두 가져온다.


7. slice()
=> subString과 같지만 음수로직을 사용할 수 있다.


8. repeat();
=> 인자로 숫자를 주어서 해당 문자열을 반복시킨다.


9. includes();
=> 해당 문자열이 존재하는지 Boolean 값으로 나타낸다.


10. startsWith();
=> 해당 문자열로 시작하는지 Boolean 메서드를 통해서 나타낸다.


11. endsWith();
=> 해당 문자열로 끝나는지 Boolean 메서드를 통해서 나타낸다.


12. valueOf()
=> 객체 문자열 값을 원시값을 바꾼다.
