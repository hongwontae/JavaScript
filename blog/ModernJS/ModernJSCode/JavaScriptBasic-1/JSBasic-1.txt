1. Script 태그
스크립트 태그의 속성 type은 모듈에서만 사용한다.
스크립트 태그의 language 속성은 스크립트 태그의 언어를 의미하는데 이젠 JS만 써서 의미없다.
src => 파일 경로, URL로 적어도 된다.
스크립트 태그의 src와 script 태그내의 코드들은 공존하지 못한다.


2. 코드 구조
문 => 어떤 작업을 수행하는 문법 구조(syntax structure)와 명령어(command)를 의미한다.
세미콜론 => 문을 끝마칠 떄 사용한다.


3. 'use strict'
최상단에 위치하면 해당 코드들은 모두 엄격모드를 따른다.
클래스와 모듈은 자동으로 useStrict 모드로 작동한다.


4. 변수와 상수
let, var => 바뀔 수 있는 변수
const => 바꿀 수 없는 변수 => 상수
=은 할당 연산자이다.


5. 자료형
string, number, boolean, bigInt, symbol, null, undefined, obg
=> 변수에 저장되는 값의 타입은 언제든지 바꿀 수 있는 언어를 동적 타입 언어라고 부른다.

number
Infinity는 어떤 숫자보다 더 큰 특수 값, 무한대(∞)를 나타낸다.
Nan => 계산 중에 에러가 발생할 떄 나오는 값 => NaN은 변하지 않음

bigInt
n를 붙이면 bigInt이다.

typeof => 자료형을 알아낼 떄 사용하는 연산자이다. 결과값은 문자열로 알려준다.


6. alert, prompt, confirm
alert => 알림창
prompt(title, default) => title을 알림창의 메시지, dafalut는 값이 없을 떄 주는 기본값
=> 값을 가져온다.
confirm(question) => 확인 버튼 true, 그 외 false를 가져온다.
=> 이들은 모달창이 떠 있는 동안 스크립트 실행을 하지 않는다.
