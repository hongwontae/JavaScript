1. 전역 공간 최소화
=> 전역 공간은 Global과 Window로 나뉩니다.
=> 브라우저 환경에서 돌아가는 경우에는 Window가 최상위입니다.
=> Node.js 환경에서는 Global이 최상위입니다.
=> 즉, 최상위 공간이 전역 공간입니다. 최상위 공간에는 브라우저 공간, Node.js 공간 둘로 분기합니다.


2. 전역 공간에서 var로 변수를 선언하면 window 환경에 변수가 추가됩니다.


** var를 사용하거나 최상위 스코프에서 함수 선언문을 사용하면 window 객체에 추가됩니다.
=> 분리된 줄 알고 사용하다가 에러를 일으킬 가능성이 높거나 기존의 window 객체를 오버라이딩하여 
에러를 일으킬 수 있습니다.
