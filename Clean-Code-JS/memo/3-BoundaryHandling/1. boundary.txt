1. min-max
1-1. 포함 여부를 결정해야 합니다. (이상, 초과/ 이하, 미만)
1-2. 변수 네이밍에 최소값, 최대값과 포함 여부를 고려하여 지어야 합니다.


2. begin-end (시작과 끝)
2-1. 시작 변수는 begin/ 끝 변수는 end 라는 단어를 포함시키면 좋습니다.


3. first-last (첫 번쨰, 마지막)


4. prefix-suffix (접두사, 접미사)
=> 접두사는 어떤 단어 앞쪽에 붙는 것을 의미 (시어머니, 시아버지, useState, useContext..) (prefix)
=> 접미사는 단어 뒤족에 붙는 것을 의미합니다. (덮개, 지우개, 마개..) (suffix)
