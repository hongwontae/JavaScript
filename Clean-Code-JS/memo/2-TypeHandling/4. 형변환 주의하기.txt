1. 동등 연산자로 비교하면 암묵적 형변환이 발생합니다.

2. 숫자와 문자열을 + 연산자를 사용하면 문자열로 접합됩니다.

3. parseInt('', 10)
=> 문자 타입을 숫자 타입으로 전환하는데 두 번쨰 인자를 따릅니다. (10을 넣으면 10진수)

4. 명시적인 형 변환 함수
4-1. String('undefined') => 'undefined'
4-2. Number('111') => 111
4-3. Boolean([]) => false


** 사용자가 타입을 변환 => 명시적인 형 변환
** JS의 평가로 인해 변환 => 암시적인 형 변환
