1. 컴퓨터는 2진수로 데이터를 바라봅니다.


2. JS는 IEEE 754 표준을 사용합니다.
=> 컴퓨터에서 부동소수점을 표현하는 표준입니다.


3. Number.MAX_SAFE_INTEGER
=> 가장 큰 정수를 반환합니다.

4. Number.isInteger()
=> 인자의 값이 정수인지 확인합니다.

5. isNaN()
=> 주어진 값을 숫자로 변환했을 떄 NaN인지 검사하는 함수입니다.
=> 느슨한 검사를 실행합니다. isNaN(123 + '테스트') => true


6. Number.isNaN()
=> 엄격한 검사를 실행합니다.
=> 즉, 전달된 값을 형변환 하지 않고 그대로 검사합니다.
