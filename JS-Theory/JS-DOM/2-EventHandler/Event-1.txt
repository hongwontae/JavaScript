1. 이벤트 핸들러를 호출하는 방법

1-1. html 속성을 통해 핸들러를 등록하는 방법
=>  <button onclick="handleClick()">

1-2. 이벤트 핸들러 프로퍼티 방법
=> document.getElementById('btn').onclick = function(){...};
=> 해당 방식은 하나의 이벤트 핸들러만 등록할 수 있다.
=> 객체의 프로퍼티를 할당시키는 방법이기 떄문이다.

1-3. addEventListner
=> on x
=> 하나 이상의 핸들러 등록 가능
=> 등록된 순서대로 호출
=> 참조가 동일한 핸들러의 경우 하나만 등록된다.


2. Event Object
2-1. 이벤트가 발생하면 자동으로 이벤트 핸들러에게 이벤트 객체가 매개변수로 들어간다.

2-2. HTML 어트리뷰트 방식은 event로 정확히 명시해야 한다.
