1. 팩토리 함수
함수를 선언해서 함수 내부에서 객체를 생성해 반환하는 함수를 의미한다.

ex )function makeColor(r, g, b) {
  const color = {};
  color.r = r;
  color.g = g;
  color.b = b;
  color.rgb = function () {
    const { r, g, b } = this;
    return `rgb(${r}, ${g}, ${b})`;
  }

  return color;
}

const colorList = makeColor(15, 20, 25);
const purple = makeColor(35, 200, 250);

=> 생성자 함수가 존재하지 않았을 떄 사용했던 기법
=> 내부에서 객체를 임의적으로 생성하고 해당 객체에 프로퍼티와 메서드를 인위적으로 추가해서 return하는 방식이다.
=> 생성자 함수가 더 간편하고 prototype의 기능이 있기 떄문에 팩토리 함수는 이제 거의 사용하지 않는다.
